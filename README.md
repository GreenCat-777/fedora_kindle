# fedora_kindle
Fedora shell for kindle.

# Prerequsites
armhf jailbroken kindle
access to /mnt/base-us (you can use ssh for this)

# Setup
take latest release
extract it
copy fedora folder to /mnt/base-us (make sure no nested folders)
to launch use [booklet]() (drag to /mnt/us/documents) or type "sh /mnt/base-us/fedora/fedora.sh" into kterm
